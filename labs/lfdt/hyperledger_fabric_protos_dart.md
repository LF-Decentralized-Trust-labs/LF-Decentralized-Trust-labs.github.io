# Lab Name
hyperledger_fabric_protos_dart

# Short Description
dart bindings of fabric-protos for hyperledger
fabric version 2.4 and later .

# Scope of Lab
provide a low-level dart bindings for [ fabric-protos ](
https://github.com/hyperledger/fabric-protos ) .  additionally , provide
a simple sdk for reference implementation similar to
[fabric-rust-integration](./fabric-rust-integration.md) .  this lab
could be a low-level base for numerous high-level projects , including
the
[hyperledger_fabric_flutter_labs](https://github.com/ghpZ54K8ZRwU62zGVSePPs97yAv9swuAY0mVDR4/hyperledger_fabric_flutter_labs)
for high-level application scenarios .  the grpc protos generated for
dart language will be intended for use in gateway enabled peers for
hyperledger fabric version 2.4 and later .
