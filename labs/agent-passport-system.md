# Lab Name
Agent Passport System

# Short Description
Open protocol for AI agent identity: verifiable identity, scoped delegation with monotonic narrowing, and signed tamper-evident receipts for permitted and denied actions.

# Scope of Lab
AI agents increasingly act on behalf of people and organizations, but most systems cannot answer three questions: whose authority is this agent using, did that authority only narrow as it was delegated, and what did a specific decision actually allow. Agent Passport System (APS) specifies and implements verifiable agent identity, delegation chains bound to a human or organizational principal, and signed tamper-evident receipts for both permitted and denied actions. It is a natural fit for LF Decentralized Trust's work on digital identity and verifiable trust infrastructure, and is specified as IETF Internet-Draft draft-pidlisnyi-aps. Apache 2.0 throughout.
